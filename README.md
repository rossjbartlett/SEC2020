# SEC 2020

Application bootstrapped with <a href="http://example.com/" target="_blank">Create React App</a>.

### Setup

Run the server with `npm run server`. To specify the port, use `PORT=xxxx npm run server`. The default is 4000.

Run the client with `npm start`. The client runs on localhost:3000. To specify the client's connection to the server, use `REACT_APP_SERVER=host:port npm start`. The deafult server is localhost:4000.
