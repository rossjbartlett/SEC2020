# SEC 2020

Application bootstrapped with <a href="http://example.com/" target="_blank">Create React App</a>.

### Setup

After first clone, install dependencies with `npm install`.

### Run

Run the server with `npm run server`. To specify the port, use `PORT=xxxx npm run server`. The default is 4000.

Run the client with `npm start`. The client runs on localhost:3000.

To specify the client's connection to the server, use `REACT_APP_SERVER_PORT=xxxx npm start`. The default is 4000.
